Inventory Report Import

Inventory Report Format:

Upload your inventory report CSV with the following columns:

Required Columns:
  • SI No: Serial number
  • Product: Product type (Contact Lens, Lens, Sunglasses, Frame)
  • Product Code: Model code
  • Description: Product description
  • Branch Name: Location/branch
  • Quantity: Stock quantity
  • Pieces Per Box: Packaging information
  • Total Number Of Pieces: Calculated total
  • Average Unit Price: Price per unit
  • Average Tax %: Tax percentage
  • Total Purchase (Rs): Total cost
